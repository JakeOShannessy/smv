Hashing Computation Notes

This directory contains scripts and files used to verify that the hashing program hashfile creates the correct md5, sha1 and sha256 hashes.

The hashing algorithms implemented in the program hashfile are verified by computing the md5, sha1 and sha256 hashes for the file named base and comparing the result to the reference hashes saved in the file named reference. The results in reference were verified by computing the hashes for the file base and comparing them to results found using the online calculator located at http://onlinemd.com
